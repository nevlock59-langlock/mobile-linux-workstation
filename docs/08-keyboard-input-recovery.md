# Keyboard / Input Recovery

Keyboard and shortcut behavior was previously one of the hardest parts of the Android + RVNC Viewer + Xfce4 setup.

## Current status

As of recovery check, the following inputs work:

- Alphabet keys
- Number keys
- Enter
- Backspace
- Tab
- Ctrl shortcuts
- Alt+Tab
- Arrow keys
- Korean input
- Touch / soft keyboard interaction

## Notes

The current environment appears to have preserved the working keyboard/input configuration.

The exact historical fixes are not fully reconstructed yet, but the practical input layer is currently usable.

## Future note

If this environment is rebuilt on a new device, keyboard/input compatibility should be tested early because RVNC Viewer, Android input handling, and Xfce4 shortcuts may interact differently.
